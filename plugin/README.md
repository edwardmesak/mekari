# SEO Plugin Workspace

This folder contains WordPress plugin projects and release artifacts.

## Contents

- `links-manager/` - Internal/external links management plugin code

## Quick Start

1. Open the plugin folder in VS Code.
2. Work inside the plugin directory you want (`content-refresher/` or `links-manager/`).
3. Follow each plugin's own README for setup and development details.

## Notes

- Source code lives in the plugin subfolders.
- Use versioned ZIP files only for distribution/release.
